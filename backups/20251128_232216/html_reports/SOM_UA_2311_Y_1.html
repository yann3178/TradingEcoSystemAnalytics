üìù Summary

This is an intraday breakout strategy that trades once per week on a specific day. It uses swing-based volatility (GSV) to determine breakout thresholds and requires pattern confirmation before entry. The strategy looks for price to break above/below the opening price by a multiple of recent swing volatility.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Close > (Open + (GsvLong * SignalGSV)) - breakout above open by 2.8x swing volatility ‚Ä¢ UAPtnBase(PtnBaseLY,ohlcvalues) = true - Pattern 41 confirmation ‚Ä¢ UAPtnBase(PtnBaseLN,ohlcvalues) = false - Pattern 42 rejection ‚Ä¢ Day of week = OpenDay (1=Monday) ‚Ä¢ Time between 1700 and 1200 (next day) ‚Ä¢ No existing position SHORT Entries: ‚Ä¢ Close < (Open - (GsvShort * SignalGSV)) - breakout below open by 2.8x swing volatility ‚Ä¢ UAPtnBase(PtnBaseSY,ohlcvalues) = true - Pattern 39 confirmation ‚Ä¢ UAPtnBase(PtnBaseSN,ohlcvalues) = false - Pattern 23 rejection ‚Ä¢ Day of week = OpenDay (1=Monday) ‚Ä¢ Time between 1700 and 1200 (next day) ‚Ä¢ No existing position

// Works using 15/30/60 minute bars

IsStartOfSession = _OHLCMulti5(sessionStartTimeC, sessionEndTimeC, ohlcValues);
if isStartOfSession then MyEndtime = UACalcEndTime(sessionStartTimeC, sessionEndTimeC);

Input: nContracts(1), nBarsGSV(10), SignalGSV(2.8);
Input: PtnBaseLY(41), PtnBaseLN(42);
Input: PtnBaseSY(39), PtnBaseSN(23);
Input: MyStartTrade(1700), MyEndTrade(1200);
Input: OpenDay(1), CloseDay(5);
Input: MyStop(2500), MyProfit(5000);

vars: GsvLong(0), GsvShort(0), OldBar(0);

if (OldBar <> CurrentBar) then begin
	OldBar = CurrentBar;
	GsvLong = _LG_SwingLong(nBarsGSV);
	GsvShort = _LG_SwingShort(nBarsGSV);
end;

If marketposition = 0 and DayOfWeek(d) = OpenDay and tw(MyStartTrade,MyEndTrade) then begin

	if Close > (Open + (GsvLong * SignalGSV)) and UAPtnBase(PtnBaseLY,ohlcvalues) and UAPtnBase(PtnBaseLN,ohlcvalues)=false then buy("LE") nContracts contracts next bar at market;
	if Close < (Open - (GsvShort * SignalGSV)) and UAPtnBase(PtnBaseSY,ohlcvalues) and UAPtnBase(PtnBaseSN,ohlcvalues)=false then sellshort("SE") nContracts contracts next bar at market;
	
end;

Setstopcontract;
If MyStop > 0 then setstoploss(MyStop);
If MyProfit > 0 then setprofittarget(MyProfit);

if DayOfWeek(d) = CloseDay and Time=MyEndTime then begin
	buytocover("SX_EOSess") next bar at market;
	sell("LX_EOSess") next bar at market;
end;
