üìù Summary

This is a counter-trend strategy for ES futures on 15-minute bars that waits for an initial time period, then enters trades when price breaks beyond calculated range levels. It uses session high/low ranges to define entry levels and combines pattern recognition with time-based filters for mean reversion trades.

//Input

input: myrangeportion_LONG(0.6), myrangeportion_SHORT(0.8);
input: MyPtnLY(41), MyPtnSY(26);

//definitions
slb = sessionlastbar;

if slb[1] then mycounter = 0;
mycounter = mycounter +1;

if mycounter <= myduration then begin
	Myentrylevellong = lowS(0) - (highS(0)-lowS(0))*myrangeportion_LONG;
	Myentrylevelshort = highS(0) + (highS(0)-lowS(0))*myrangeportion_SHORT;
End;

if time = endtime then ExitID = 0;

//conditions

	if openpositionprofit > MyCheckGain then ExitID = 1;
