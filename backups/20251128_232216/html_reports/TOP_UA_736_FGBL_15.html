üìù Summary

This is a breakout strategy that enters long when price breaks above the N-session highest high and short when breaking below the N-session lowest low. The strategy uses ADX filtering (must be below threshold), multiple pattern filters, and operates within specific time windows with pause periods.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Price breaks above N-session (default 3) highest high ‚Ä¢ Within time window (11:00-21:00) but not during pause (17:00-19:00) ‚Ä¢ ADX < 40 (low trending environment) ‚Ä¢ PatternNeutralFast(55) = true (neutral pattern) ‚Ä¢ PatternNeutralFast(45) = false (avoid specific neutral pattern) ‚Ä¢ PatternDirectionalFast(+52) = true (bullish directional pattern) ‚Ä¢ PatternDirectionalFast(+38) = false (avoid specific bullish pattern) ‚Ä¢ Not on skip session day for longs SHORT Entries: ‚Ä¢ Price breaks below N-session (default 3) lowest low ‚Ä¢ Same time, ADX, and neutral pattern conditions as longs ‚Ä¢ PatternDirectionalFast(-52) = true (bearish directional pattern) ‚Ä¢ PatternDirectionalFast(-38) = false (avoid specific bearish pattern) ‚Ä¢ Not on skip session day for shorts (day 5)

üö™ Exit Conditions

üîç Pattern Details

	MyStartTime(1100), MyEndTime(2100), MyStartPause(1700),MyEndPause(1900), //Time window

	PtnNeutNo(45), ptnDirYes(52), ptnDirNo(-38), //Directional patterns (optimize from -52 to +52. +/-52 = true, 0 = false)
	SkipSessL(-1), SkipSessS(5), //Session of the week to skip
	maxdaysintrade(0), flatTime(1630), //Max days in position (1=intraday, 0=no limit)
	MyStop(1000), MyBreakeven(1000), MyProfit(4000); //Stoploss, breakeven, takeprofit (cash amount per contract)
	
var: hh(0), ll(0), OKL(true), OKS(true), DaysInTrade(0), SoW(0), mp(0), SessAcross2Days(false), FlatTimeFixed(0);

//la parte sottostante serve solo per scrupolo e per coprire tutti i possibili sottostanti, sia quelli a cavallo tra due sessioni che quelli che non lo sono:
once begin
	SessAcross2Days = SessBegin>SessEnd; //Verifies if the session is across 2 calendar days (cosa che succede per la maggior parte dei sottostanti)
	
	//Determines the correct flat time according to the session type; 
	if SessAcross2Days then FlatTimeFixed = SessBegin
	else FlatTimeFixed = SessEnd;
	if SessBegin=0 and SessEnd=0 then FlatTimeFixed = 2359;
end;

//iniziamo la parte centrale del codice:

If isStartOfSession then begin
	//ADX Calculation using Michael's formula;
	//noi non usiamo la formula classica di MC per calcolare l'ADX per due motivi: primo stiamo lavorando intraday e vogliamo l?ADX delle ultime N sessioni; 
	//secondo potremmo scegliere una custom session;
	adxVal = iADXOnArray(ADXLen, ohlcValues[5], ohlcValues[6], ohlcValues[7], ohlcValues[9], ohlcValues[10], ohlcValues[11], adxCalcValues)*100;
	
	//N-sessions Highest High/Lowest Low calculation
	//qui inizia il loop: se il numero sessioni e' pari ad 1 assegna alla variabili il max ed il minimo di ieri,
	// altrimenti va in loop fino a trovare il massimo ed il minimo su N sessioni;
	hh = -99999999999;	ll = 99999999999;
	if nSess>1 then begin
		for value90 = 1 to nSess begin
			hh = maxlist(hh,ohlcvalues[1+value90*4]);
			ll = minlist(ll,ohlcvalues[2+value90*4]);
		end;
	end
	else if nSess=1 then begin
		hh = ohlcValues[5];
		ll = ohlcValues[6];
	end;
	
	//Reset permission to work long and short at every session reset (N.B. if you go overnight OKL/OKS will become false at next bar!)
	OKL = true;	OKS = true;
	
	//days in position counter increases. Questo serve se volessimo usare l'uscita a tempo dopo un tot di sessioni; 
	if marketposition<>0 then DaysInTrade = DaysInTrade+1; 
	
	//Actual session count within the trading week
	//sow = session of week - e' una finezza da codice avanzato che permette di gestire il giorno di skip session nel caso di sottostante a cavallo di due sessioni; 
	//infatti la variabile SoW la troviamo piu' in basso nel momento nelle condizioni di entrata;
	if SessAcross2Days then sow = dayofweek(d)+1
	else sow = dayofweek(d);
end;

//qui confrontiamo il massimo ed il minimo con quelli della sessione odierna nel caso in cui volessimo includere anche la sessione odierna nel confronto:
if levIncludeSess0=1 then begin
	hh = maxlist(hh,h);
	ll = minlist(ll,l);
end;

//qui troviamo un secondo pattern neutral yes; si puo' usare ma attenzione all'overfittig;

if tw(MyStartTime,MyEndTime) and (t<MyStartPause or t>MyEndPause) and adxVal<adxth and PatternNeutralFast(PtnNeutYes,ohlcvalues) and 
PatternNeutralFast(PtnNeutYes2,ohlcvalues) and PatternNeutralFast(PtnNeutNo,ohlcvalues)=False then begin

	if okl and PatternDirectionalFast(+ptnDirYes,ohlcvalues) and PatternDirectionalFast(+ptnDirNo,ohlcvalues)=false and sow<>SkipSessL then
	buy("LE") mycontracts contracts next bar hh stop;

	sellshort("SE") mycontracts contracts next bar ll stop;
end;

//qui di seguito resettimao la variabile contatore DaysInTrade nel caso in cui avessimo reversato la posizione 

mp = marketposition;
if mp<>mp[1] and mp<>0 then DaysInTrade = 1;   

//qui di seguito blocchiamo l'operativa' long e short se siamo gia' in posizione:

if marketposition=-1 then oks = false;

	if t>=flatTime and t<FlatTimeFixed then begin

		buytocover("SX_MaxDays") next bar market;
	end;
end;

setstopcontract;
