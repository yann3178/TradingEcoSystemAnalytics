üìù Summary

This is a pattern-filtered Donchian breakout system that enters long/short positions when price breaks the 150-period Donchian channel extremes. The strategy uses four pattern conditions (two neutral and two directional) to filter entries and includes session-based time limits.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Market position = 0 ‚Ä¢ Time within session window (1800-1700) ‚Ä¢ PatternNeutralFast(26) = TRUE (neutral pattern confirmation) ‚Ä¢ PatternNeutralFast(6) = FALSE (neutral pattern rejection) ‚Ä¢ PatternDirectionalFast(+52) = TRUE (directional pattern for long) ‚Ä¢ PatternDirectionalFast(+53) = FALSE (directional pattern rejection for long) ‚Ä¢ Buy at Donchian Upper (150-period high) stop SHORT Entries: ‚Ä¢ Market position = 0 ‚Ä¢ Time within session window (1800-1700) ‚Ä¢ PatternNeutralFast(26) = TRUE (neutral pattern confirmation) ‚Ä¢ PatternNeutralFast(6) = FALSE (neutral pattern rejection) ‚Ä¢ PatternDirectionalFast(-52) = TRUE (directional pattern for short) ‚Ä¢ PatternDirectionalFast(-53) = FALSE (directional pattern rejection for short) ‚Ä¢ Sell at Donchian Lower (150-period low) stop

//VARIABLES
var: MyDonchianUpper(0), MyDonchianLower(0);

//DEFINITIONS
MyDonchianUpper = HighestFC(high,DonchianPeriod);
MyDonchianLower = LowestFC(low,DonchianPeriod);


//here below we use patterns in a mirrorerd setup so that the same or opposite condition appliy to long or short
// to do so either neutral patterns or mirrored directional patterns are used
If mp=0 and TW(MyStartTrade,MyendTrade)//time window
and PatternNeutralFast(PtnNeutYes, ohlcValues) and PatternNeutralFast(PtnNeutNo, ohlcValues)=false then begin

	If PatternDirectionalFast(+PtnDirYes, ohlcValues) and PatternDirectionalFast(+PtnDirNo, ohlcValues)=False 
	then buy MyContracts contracts next bar at MyDonchianUpper stop;
	
	If PatternDirectionalFast(-PtnDirYes, ohlcValues) and PatternDirectionalFast(-PtnDirNo, ohlcValues)=False 
	then sellshort MyContracts contracts next bar at MyDonchianlower stop;
