üìù Summary

This is a session-based bias strategy that trades different patterns depending on the day of the week. It uses Thursday sessions for long bias and Sunday sessions for short bias, combined with pattern recognition and breakout confirmation using hourly highs/lows.

LONG Entries: ‚Ä¢ Must be Thursday (dayofweek = 4) ‚Ä¢ Must be within trading window (20:00 to 07:00) ‚Ä¢ PatternFast(68) condition must be TRUE (closed1 < closed2) ‚Ä¢ PatternFast(153) condition must be FALSE (invalid pattern number) ‚Ä¢ High[1] > High[4] (1-bar high breaks 4-bar high) SHORT Entries: ‚Ä¢ Must be Sunday (dayofweek = 0) ‚Ä¢ Must be within trading window (20:00 to 07:00) ‚Ä¢ PatternFast(13) condition must be TRUE (body5d < 1 * (highd5-lowd1)) ‚Ä¢ PatternFast(153) condition must be FALSE (invalid pattern number) ‚Ä¢ Low[1] < Low[4] (1-bar low breaks 4-bar low)
