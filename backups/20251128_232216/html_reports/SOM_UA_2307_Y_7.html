üìù Summary

This is a session-based breakout strategy that trades breakouts of previous session highs/lows when ADX is below 50 (indicating low trend strength). It uses pattern filters to confirm entry conditions and includes comprehensive risk management with stop loss, breakeven, and profit targets.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Current time within trade window (20:00 to 15:00) ‚Ä¢ ADX < 50 (low trend strength filter) ‚Ä¢ Less than 2 trades today ‚Ä¢ PatternFast(4): body1d < 0.75 * range1d (small body pattern present) ‚Ä¢ PatternFast(2): body1d < 0.25 * range1d (very small body pattern absent) ‚Ä¢ Buy stop at high of session 1 day ago SHORT Entries: ‚Ä¢ Same time and ADX conditions as long ‚Ä¢ PatternFast(41): (opend0-lowd0) > ((opend1-lowd1)*0.5) (current gap down > 50% of previous gap) ‚Ä¢ PatternFast(42): (opend0-lowd0) > ((opend1-lowd1)*1) (current gap down > previous gap) - must be absent ‚Ä¢ Sell short stop at low of session 1 day ago

inputs:
  myContracts(1),
  sessionStart(1800),
  sessionEnd(1700),
  tradeStart(2000),

  adxValue(0),
  longLevel(0),
  shortLevel(0),
  mp(0),
  sessionsInTrade(0);

arrays:
  ohlc[23](0),
  adxValues[3](0);

isStartOfSession = _OHLCMulti5(sessionStart, sessionEnd, ohlc);
longLevel = ohlc[1 + 4 * sessionsAgo];
shortLevel = ohlc[2 + 4 * sessionsAgo];
mp = marketposition;

if isStartOfSession
then begin
  adxValue = iADXOnArray(5, ohlc[5], ohlc[6], ohlc[7], ohlc[9], ohlc[10], ohlc[11], adxValues) * 100;

  if mp <> 0 then sessionsInTrade = sessionsInTrade + 1;
end;

if tw(tradeStart, tradeEnd)
   and adxValue < adxThreshold
   and EntriesToday(d) < maxTrades
then begin
  if PtnBaseSA2(ptnLongY, ohlc)
     and not PtnBaseSA2(ptnLongN, ohlc)
  then begin
    buy("LE") myContracts contracts next bar at longLevel stop;
  end;

  if PtnBaseSA2(ptnShortY, ohlc)
     and not PtnBaseSA2(ptnShortN, ohlc)
  then begin
    sellshort("SE") myContracts contracts next bar at shortLevel stop;
  end;
end;

if mp <> mp[1] and mp <> 0 then sessionsInTrade = 1;

if maxSessionsInTrade > 0 and sessionsInTrade >= maxSessionsInTrade then begin
  setexitonclose;
  
  if time = flatTime then begin
    sell("LX_MaxDays") myContracts contracts next bar at market;
    buytocover("SX_MaxDays") myContracts contracts next bar at market;
  end;
end;

setstopcontract;
if stopLoss > 0 then setstoploss(stopLoss);
if breakEven > 0 then setbreakeven(breakEven);
if takeProfit > 0 then setprofittarget(takeProfit);
