üìù Summary

This is a sophisticated intraday breakout strategy that uses multiple pattern confirmations and directional filters. It trades breakouts of previous session highs/lows with extensive pattern filtering and risk management for intraday sessions.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Buy at previous session high (myle) on stop order ‚Ä¢ Must be within trading time window ‚Ä¢ Maximum 4 trades per day not exceeded ‚Ä¢ PtnNeutYes pattern TRUE and PtnNeutNo pattern FALSE ‚Ä¢ PtnLY pattern TRUE and PtnLN pattern FALSE ‚Ä¢ Positive PtnDirYes pattern TRUE and positive PtnDirNo pattern FALSE SHORT Entries: ‚Ä¢ Sell short at previous session low (MySe) on stop order ‚Ä¢ Must be within trading time window ‚Ä¢ Maximum 4 trades per day not exceeded ‚Ä¢ PtnNeutYes pattern TRUE and PtnNeutNo pattern FALSE ‚Ä¢ PtnSY pattern TRUE and PtnSN pattern FALSE ‚Ä¢ Negative PtnDirYes pattern TRUE and negative PtnDirNo pattern FALSE

üíª Strategy Code

input: MyContracts(1),
	sessionStartTimeA(1800), sessionEndTimeA(1700),
	MaxTradesPerDay(4),
	PtnDirYes(52), PtnDirNo(8), PtnNeutYes(4), PtnNeutNo(56),
	PtnLY(38), PtnLN(42), PtnSY(41), PtnSN(42),
	MyStop(1625), MyProfit(2050), ID(0), MyTrigger(2),
	MyStartTrade(2100), MyEndTrade(900), CloseAtTime(2500), sesstest(0);

vars: highd0(0),lowd0(0), mybar(0), myle(99999),mySE(0),highd1(0),lowd1(0), MyEndTime(0), 
	endsession(0), calcstart(0), calcend(0),timewindow(false);

//end of session setting////////////////////////
if sessionEndTimeA >= 2400 then endsession = sessionStartTimeA
else endsession = sessionEndTimeA;
////////////////////////////////////////////////

//>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>	Michael OHLC Calculation begin
array: ohlcValues[23](0);
var: isStartOfSession(false);
isStartOfSession = _OHLCMulti5(sessionStartTimeA, endsession, ohlcValues);
//>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>	Michael OHLC Calculation end

highd0 = ohlcValues[1]; lowd0 = ohlcValues[2];  
highd1 = ohlcValues[5]; lowd1 = ohlcValues[6];  

// setting of time to close positions on ID trading
if isStartOfSession then MyEndtime = UACalcEndTime(sessionStartTimeA,endsession);
if CloseAtTime <> 2500 then MyEndTime = CloseAtTime; // only if we want a specific time to close trades
////////////////////////////////////////////////

//levels choice/////////////////////////////////
if Mytrigger = 0 then begin
	myle = highs(0);//highsession(0,0);
	MySe= lows(0);//lowsession(0,0);
end;
if Mytrigger = 1 then begin
	myle = highd0;
	MySe= lowd0;
end;
if Mytrigger = 2 then begin
	myle = highd1;
	MySe= lowd1;
end;
////////////////////////////////////////////////

// setting of time window depending on a number of conditions
if sesstest = 0 then begin //during normal test we set them as inputs
	calcstart = MyStartTrade;
	calcend = MyEndTrade;
end
else begin // with sesstest = 1 we set start and end 2 hours after start and 2 hours before end of session
	calcstart = sessionStartTimeA + 200; //2 hours after session start
	if calcstart > 2359 then calcstart = calcstart - 2400;
	calcend = endsession - 200; //2 hours before session end
	if calcend < 0 then calcend = calcend + 2400;
end;
timewindow = tw(calcstart,calcend);
////////////////////////////////////////////////

// T R A D I N G ////////////////////////////////
if timewindow and EntriesToday(d)<MaxTradesPerDay and PatternNeutralFast(PtnNeutYes,ohlcvalues) and PatternNeutralFast(PtnNeutno,ohlcvalues)=false then begin
	if UAPtnBase(PtnLY,ohlcvalues) and UAPtnBase(PtnLN,ohlcvalues)=false and PatternDirectionalFast(+PtnDirYes,ohlcvalues) and PatternDirectionalFast(+ptnDirNo,ohlcvalues)=false then 
		buy MyContracts contracts next bar at myle stop;
	
	if UAPtnBase(PtnSY,ohlcvalues) and UAPtnBase(PtnSN,ohlcvalues)=false and PatternDirectionalFast(-PtnDirYes,ohlcvalues)and PatternDirectionalFast(-ptnDirNo,ohlcvalues)=false then 
		sellshort MyContracts contracts next bar at MySe stop;
end;

setstopcontract;
if MyStop> 0 then setstoploss(mystop);
if MyProfit > 0 then setprofittarget(Myprofit);

if ID = 0 then begin
	if MyTrigger = 0 then setexitonclose;
	if mytrigger >= 1 and time =MyEndTime then begin
		buytocover("EOSessSX") next bar at market;
		sell("EOSessLX") next bar at market;
	end;
end;
