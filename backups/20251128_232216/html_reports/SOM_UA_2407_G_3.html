üìù Summary

Session-based breakout strategy that trades breakouts from previous session's range (17:00-16:00) using ATR buffers. Uses dynamic ATR-based stop losses with asymmetric parameters for long and short trades.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Current high > previous session high + (3.0 * 20-period ATR) ‚Ä¢ Time between 09:00 and 14:00 ‚Ä¢ Maximum 2 entries per day ‚Ä¢ Re-entry only if previous same-day trade was losing SHORT Entries: ‚Ä¢ Current low < previous session low - (3.5 * 20-period ATR) ‚Ä¢ Time between 09:00 and 14:00 ‚Ä¢ Maximum 2 entries per day ‚Ä¢ Re-entry only if previous same-day trade was losing

üö™ Exit Conditions
