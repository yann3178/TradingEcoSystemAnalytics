üìù Summary

This strategy identifies critical levels from Data2 (higher timeframe) and trades breakouts of rounded levels on the primary timeframe. It uses pattern confirmation to filter entries and includes strict time-based position management.

inputs: roundTo(50),MyContracts(1),
	   timeSessStart(1700), timeSessEnd(1600), timeOpenStartHr(1800), timeOpenEndHr(1500), timeOpenStartMin(0), timeOpenEndMin(15),
	   maxBarsOpenLong(0), maxBarsOpenShort(28), maxTradesPerDay(1), sl(1100), tp(0), be(0),
	   patLongYes(13), patLongNo(78), patShortYes(150), patShortNo(34);

arrays: ohlcValues[23](0), ohlcValuesD2[23](0);

variables: isStartSess(true), lvlAbove(0), lvlBelow(0), counterTrade(0), mp(0), criticLevel(0), dummy(true);

isStartSess = _OHLCMulti5(timeSessStart, timeSessEnd, ohlcValues);
dummy= _OHLCMulti5_d2(timeSessStart, timeSessEnd, ohlcValuesD2);
criticLevel = ohlcValuesD2[0];

if isStartSess = true then
	counterTrade = 0;
mp = marketposition;
if (mp <> 0) and (mp <> mp[1]) then
	counterTrade = counterTrade + 1;

// rounded levels
lvlAbove = roundTo*ceiling(criticLevel/roundTo);
lvlBelow = roundTo*floor(criticLevel/roundTo);

// entry positions
if tw(timeOpenStartHr + timeOpenStartMin, timeOpenEndHr + timeOpenEndMin) then
begin
	// long entry condition
	if (PatternFast(patLongYes, ohlcValues)) and (PatternFast(patLongNo, ohlcValues) = false) and
	   (counterTrade < maxTradesPerDay) and (close data2 crosses above lvlAbove) then
		buy MyContracts contracts next bar market;
	// short entry condition
	if (PatternFast(patShortYes, ohlcValues)) and (PatternFast(patShortNo, ohlcValues) = false) and
	   (counterTrade < maxTradesPerDay) and (close data2 crosses below lvlBelow) then
		sellshort MyContracts contracts next bar market;
end;

// exit conditions
if (barssinceentry >= maxBarsOpenLong) and (maxBarsOpenLong > 0) and (marketposition = 1) then
	sell next bar at market;
if (barssinceentry >= maxBarsOpenShort) and (maxBarsOpenShort > 0) and (marketposition = -1) then
	buytocover next bar at market;

setexitonclose;

setstopcontract;
if sl > 0 then
	setstoploss(sl);
if tp > 0 then
	setprofittarget(tp);
if be > 0 then
	setbreakeven(be);
