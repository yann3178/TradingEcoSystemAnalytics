üìù Summary

This is a Donchian Channel breakout strategy that enters long positions when price breaks above the 24-period high and short positions when price breaks below the 24-period low. The strategy uses pattern-based filters to confirm entry conditions and includes comprehensive risk management with stops, targets, and time-based exits.

input: MyPtnLY(18), MyPtnLN(32), MyPtnSY(7), MyPtnSN(30);
input: StartTrade(500), EndTrade(1400);
input: DonchianPeriods(24);
input: MyProfit(3800), MyStop(2100), MyBarsExit(132), MyBreakEven(2600);
input: MyNoLongDay(0), MyNoShortDay(2);
input: MySize(1);

var: slb(false), mybar(0), UpperChannel(0), LowerChannel(0);

slb = sessionlastbar;
UpperChannel = HighestFC(H, DonchianPeriods);
LowerChannel = LowestFC(L, DonchianPeriods);

if slb[1] then mybar = 0;
mybar = mybar +1;

if mybar > 1 and (Time >= StartTrade and Time < EndTrade) then begin
       if PtnBaseSA(MyPtnLY)=True and PtnBaseSA(MyPtnLN)=False and dayofweek(date) <> MyNoLongDay then buy MySize contracts next bar UpperChannel stop;
       if PtnBaseSA(MyPtnSY)=True and PtnBaseSA(MyPtnSN)=False and dayofweek(date) <> MyNoShortDay then sellshort MySize contracts next bar LowerChannel stop;
end;

if BarsSinceEntry >= MyBarsExit then begin
     sell next bar at market;
     buytocover next bar at market;
end;

setstopcontract;
if MyStop <> 0 then setstoploss(MyStop);
if MyProfit <> 0 then setprofittarget(MyProfit);
if MyBreakEven <> 0 then setbreakeven(MyBreakEven);
