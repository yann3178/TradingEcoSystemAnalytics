üìù Summary

This is a Donchian Channel breakout strategy that trades breakouts of 70-period highs/lows. The strategy uses pattern filtering to enter only when specific market conditions are met, including neutral and directional patterns derived from session OHLC data.

üéØ Entry Conditions

LONG Entries: ‚Ä¢ Price breaks above 70-period Donchian upper channel ‚Ä¢ Within trading time window (04:00-11:00, excluding 12:00-11:00 pause) ‚Ä¢ Not Friday (Skipdaylong=5) ‚Ä¢ Not April trading month ‚Ä¢ PatternNeutralFast(53) is true AND PatternNeutralFast(1) is false ‚Ä¢ PatternDirectionalFast(+9) is true AND PatternDirectionalFast(+18) is false ‚Ä¢ Less than 1 entry today SHORT Entries: ‚Ä¢ Price breaks below 70-period Donchian lower channel ‚Ä¢ Same time and pattern conditions as long ‚Ä¢ No day-of-week restriction for shorts (Skipdayshort=0) ‚Ä¢ PatternDirectionalFast(-9) is true AND PatternDirectionalFast(-18) is false

Input:StartSessionTimeC(1800),EndSessionTimeC(1700);
array:ohlcvalues[23](0);

var:isstartofsession(false);
isstartofsession = _OHLCMulti5(StartSessionTimeC,EndSessionTimeC,ohlcvalues);

Input:Mysize(1),Maxentriestoday(1),DonchianPeriod(70),Skipdaylong(5),Skipdayshort(0);
Input:StartTrade(400),EndTrade(1100),StartPause(1200),EndPause(1100);
Input:PtnNeutYes(53),PtnNeutNo(1),PtnDirYes(9),PtnDirNo(18);
Input:MaxDaysinTrade(1),Exittime(1645),SL(1800),BE(0);

Var:Timewindow(false),MP(0),Daysintrade(0),Upperchannel(0),Lowerchannel(0);

Upperchannel = Highest(h,DonchianPeriod);
Lowerchannel = Lowest(l,DonchianPeriod);

if StartTrade > EndTrade then Timewindow = time > StartTrade or time < EndTrade
else if StartTrade < EndTrade then Timewindow = time > StartTrade and time < EndTrade;

if EntriesToday(d) < Maxentriestoday and Timewindow and (time < StartPause or time > EndPause) and month(d) <> 4 and PatternNeutralFast(PtnNeutYes,ohlcvalues) and PatternNeutralFast(PtnNeutNo,ohlcvalues) = false then begin
	if dayofweek(d) <> Skipdaylong and PatternDirectionalFast(+PtnDirYes,ohlcvalues) and PatternDirectionalFast(+PtnDirNo,ohlcvalues) = false then buy ("LE") Mysize contract next bar at Upperchannel stop;
	if dayofweek(d) <> Skipdayshort and PatternDirectionalFast(-PtnDirYes,ohlcvalues) and PatternDirectionalFast(-PtnDirNo,ohlcvalues) = false then sellshort ("SE") Mysize contract next bar at Lowerchannel stop;
end;

MP = marketposition;
if MP <> MP[1] and MP <> 0 then Daysintrade = 1;
if date <> date[1] and MP <> 0 then Daysintrade = Daysintrade + 1;

if MaxDaysinTrade > 0 and Daysintrade >= MaxDaysinTrade and time = Exittime then begin 
	if MP >= 1 then sell("LX") Mysize contract next bar at market;
	if MP <= -1 then buytocover ("SX") Mysize contract next bar at market;
end;

setstopcontract;
if SL > 0 then setstoploss(SL);
if BE > 0 then setbreakeven(BE);
